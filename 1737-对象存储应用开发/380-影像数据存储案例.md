## 影像数据元数据存储设计

| 字段名    | 字段名注释                        |
| ----------- | ----------------------------- |
| name    | 文件名                        |
| filelength  | 上传的文件大小                |
| lobid       | lobid是文件上传后对应的唯一id |
| putDate  | lob上传时间                   |

元数据包含文件名、文件的lobid、上传日期，大小 格式这些信息在上传lob时同步获得并写入到元数据表中.

## 影像数据存储

所有影像数据通过lob上传，作为lob对象保存在集群中
| 字段名    | 字段名注释                        |
| ---------------- | ------------------------ |
| Size             | lob对象大小              |
| Oid              | 集群自动生成的唯一标识符 |
| CreateTime       | 创建时间                 |
| ModificationTime | 修改时间                 |
| Available        | 是否可用                 |

        //获取对象元数据
